# 云端 Web 测试工具开发计划

## 项目背景

**目标**：开发 Web 版原型程序，功能完全移植本地 Windows GUI 原型程序，调用云端叙事引擎 API

**云端环境**：
- 云端引擎 API：http://8.134.212.37:80
- 服务器：8.134.212.37

**技术栈**：
- 前端：HTML + CSS + JavaScript（Web 界面）
- 后端：Flask（Web 服务，调用云端引擎 API）
- 引擎：云端叙事引擎（已部署）

---

## 任务拆分

### Task 1: Flask Web 服务基础框架搭建
- 创建 Flask 应用基础结构
- 配置静态文件目录
- 实现主页路由（/）
- 配置云端引擎 API 地址

### Task 2: 前端页面 - 对话界面
- 角色选择下拉框
- 对话窗口（显示历史消息）
- 消息输入框
- 发送按钮
- 六轴状态显示区域
- 事件卡显示区域

### Task 3: 前端页面 - 调试标签页
- 四个调试标签页：Perception / Director / Predictor / Performer
- 每个标签页显示【输入】和【输出】两个窗口
- 输入窗口：发送给 LLM 的完整 prompt
- 输出窗口：LLM 返回的原始内容

### Task 4: 后端 API - 引擎对接
- POST /api/start - 初始化引擎
- POST /api/chat - 对话接口
- GET /api/state - 获取状态
- 会话管理（支持多用户）

### Task 5: 前端交互逻辑
- 发送消息逻辑
- 轮询获取对话结果
- 对话历史显示
- 六轴实时更新
- 事件卡显示

### Task 6: 测试与调试
- 本地测试
- 云端部署测试
- 调试日志系统
