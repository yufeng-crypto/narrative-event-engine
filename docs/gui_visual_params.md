# GUI 控件视觉参数文档

> 本文档列出所有 GUI 控件的视觉表现参数，方便优化页面展示
> 修改参数后直接在文档中更新，我会按新参数修改代码

---

## 1. 窗口整体

| 参数 | 当前值 | 说明 |
|------|--------|------|
| 布局 | PanedWindow (HORIZONTAL) | 左侧聊天区 + 右侧调试区 |
| 边距 | padx=10, pady=10 | 主容器边距 |

## 2. 顶部栏

### 2.1 角色选择

| 参数 | 当前值 | 说明 |

| state | readonly | 只读模式 |
| 标签 | "角色:" | 固定文字 |

| 标签 | "模型:" | 固定文字 |

## 7. 布局参数汇总

### padx 值

| 位置 | padx 值 |
|------|---------|
| 角色选择 | 5 |
| 模型选择 | (20, 5) |
| 按钮 | 5 |
| 主容器 | 10 |
| 轴行 | 5 |
| 进度条 | 5 |

### pady 值

| 位置 | pady 值 |
|------|---------|
| 主容器 | 10 |
| 输入框 | (0, 5) |
| 标题 | (0, 5) 或 (5, 5) |
| 轴容器 | (0, 10) |
| 轴行 | 2 |
